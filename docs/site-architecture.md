# Sports Raise Site / Product Architecture

## Public marketing site

Purpose: explain Sports Raise to campaign operators and convert early pilots.

Pages/sections:
- Landing page
- How it works
- Campaign examples / pilot campaigns
- Sharing toolkit
- Rewards / prize pool
- Impact / trust
- Contact / start pilot

## Campaign operator experience

Purpose: let organizers create, launch, promote, and manage a campaign.

Core modules:
- Account / organization profile
- Campaign builder
- Cause story editor
- Campaign rules setup
- Game-day calendar
- Reward/prize pool setup
- Sharing toolkit generator
- AI video asset generator
- Impact update publisher
- Admin dashboard

## Public campaign page

Purpose: make the campaign/cause the star while clearly showing Sports Raise powers the experience.

Core modules:
- Campaign hero: cause, story, goal, CTA
- Donation flow
- Non-cash credits explanation
- Current/upcoming picks/challenges
- Prize pool / donated rewards
- Leaderboard or participation status
- Impact updates
- One-click share buttons
- Short campaign video
- Powered by Sports Raise lockup

## Contributor/fan experience

Purpose: let supporters donate, receive credits, participate in game-day challenges, and follow impact.

Core modules:
- Donate / contribute
- Receive non-cash credits
- Make picks / join challenges
- Track entries/leaderboard
- View rewards/prize pool
- Re-up / contribute again
- Share campaign
- See impact updates

## Sharing toolkit

Purpose: make campaign promotion effortless.

Assets:
- Email templates
- SMS copy
- Facebook posts
- Instagram captions/stories/reels prompts
- TikTok captions/video scripts
- LinkedIn posts for nonprofit/admin audiences
- Trackable links
- QR codes
- Embeddable widgets/buttons
- AI-generated 15s and 30s campaign videos

## AI video asset system

Initial version:
- Generic Sports Raise explainer video
- Campaign-specific video script generation
- Default prize pool language if dynamic prize data is not ready

Future version:
- Dynamic video generation using campaign story, prize pool, upcoming games/challenges, and impact milestones.

## Admin / operations

Needs:
- Campaign review and approval
- Prize/reward verification
- Challenge/rules configuration
- Donation/credit audit trail
- Supporter/contact management
- Content moderation
- Compliance checklist
- Analytics and reporting

## Key open questions

- Donation processor and compliance path
- Credit issuance rules
- Reward/prize eligibility rules
- Challenge scoring source/API
- Whether fan contributor funnel becomes separate site/URL
- What data is needed for early pilot campaigns
