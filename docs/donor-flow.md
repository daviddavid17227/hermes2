# Donor / Contributor Flow

## Goal

Make it simple for a supporter to contribute to a campaign, understand non-cash credits, participate in game-day challenges, and return later.

## Flow

1. Arrive on campaign page from share link/social/email.
2. Understand the cause and campaign goal.
3. See how Sports Raise participation works.
4. Donate to the campaign.
5. Receive non-cash credits.
6. Use credits for picks/challenges.
7. View leaderboard/reward eligibility.
8. Share the campaign.
9. Receive impact updates and reasons to come back.

## UX principles

- Cause first, mechanics second.
- Credits must be explained clearly and safely.
- Rewards should create excitement without overtaking the mission.
- Sharing should be available immediately after donation.
- Mobile flow must be primary.
