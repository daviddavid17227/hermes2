# Rewards / Prize Pool Requirements

## Goal

Use donated rewards, memorabilia, sponsor items, and Sports Raise starter prizes to drive campaign engagement while keeping the mission central.

## Reward sources

- Campaign creators
- Local businesses
- Sponsors
- Athletes/teams
- Memorabilia donors
- Sports Raise starter items for early campaigns

## Reward data

- Name
- Description
- Source/donor
- Estimated value if appropriate
- Image
- Eligibility rules
- Challenge/drop association
- Fulfillment owner
- Status: proposed, verified, live, awarded, fulfilled

## UX principles

- Reward pool should feel exciting and credible.
- Avoid cash-out framing.
- Make donated/sponsor source visible when useful.
- Tie rewards back to campaign impact and participation.
