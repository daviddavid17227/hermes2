# Admin / Operations Requirements

## Goal

Give Sports Raise internal operators enough control to launch and manage pilot campaigns safely.

## Admin capabilities

- Review campaign applications
- Approve/edit campaign copy
- Verify reward/prize pool items
- Configure non-cash credit rules
- Configure picks/challenges
- Review sharing assets
- Manage campaign status: draft, review, live, paused, completed
- Publish impact updates
- Monitor donations/credits/challenge participation
- Export campaign reports

## Pilot checklist

Before a campaign goes live:
- Campaign story complete
- Donation flow tested
- Credit explanation clear
- Prize pool verified
- Rules/eligibility documented
- Sharing toolkit generated
- Campaign video/script approved
- Mobile page QA complete
- Compliance/legal review complete where needed
