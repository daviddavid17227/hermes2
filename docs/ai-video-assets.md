# AI Video Assets Requirements

## Goal

Create short, polished videos that explain how a Sports Raise campaign works and help each campaign spread across social channels.

## Asset types

- Sports Raise platform explainer: 15s
- Sports Raise platform explainer: 30s
- Campaign-specific video: 15s vertical
- Campaign-specific video: 30s vertical
- Prize pool highlight clip
- Game-day challenge teaser
- Impact milestone update

## Story beats

1. Show the cause/campaign.
2. Show supporters donating and receiving non-cash credits.
3. Show game-day picks/challenges.
4. Show donated rewards/prize pool.
5. End with impact and share CTA.

## MVP approach

- Write reusable scripts first.
- Use campaign data to fill name, cause, prize examples, and CTA.
- If prize pool data is not ready, use default Sports Raise starter reward language.
- Generate storyboard frames/prompts before production video.

## Future dynamic inputs

- Campaign name
- Cause story
- Campaign goal
- Prize pool items
- Upcoming games/challenges
- Impact milestones
- Brand media uploaded by operator
