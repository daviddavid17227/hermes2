# Campaign Builder Requirements

## Goal

Let a campaign operator create a dedicated campaign page quickly while making the campaign/cause the star.

## Inputs

- Campaign name
- Cause/organization/person being supported
- Campaign story
- Goal amount or impact goal
- Dates / campaign calendar
- Sports focus / season / teams / game-day moments
- Prize/reward pool items
- Rules and eligibility
- Supporter audience
- Media assets: photos, logo, existing videos

## Outputs

- Public campaign page
- Donation flow copy
- Non-cash credit explanation
- Picks/challenges setup draft
- Reward/prize pool display
- Share templates
- Trackable links / QR code
- Video script or generated short video asset

## UX principles

- Campaign setup should feel guided, not like a blank CMS.
- Default copy/templates should be available immediately.
- Operators should be able to preview the campaign page before launch.
- Sports Raise brand is visible but secondary to the campaign.
